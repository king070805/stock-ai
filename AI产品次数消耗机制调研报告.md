# AI产品次数消耗机制调研报告

> 调研日期：2026年6月7日
> 调研方式：网络公开信息检索与官方文档分析

---

## 一、ChatGPT（OpenAI）

### 1. 免费用户次数限制
- **GPT-5 模型**：每 5 小时最多 **10 条消息**
- **GPT-5 Thinking 模式**：每天仅 **1 条**
- 超额后自动切换为 **GPT-5 mini**（无限次，对标 GPT-4o-mini）

### 2. 用完后如何提示
- **自动降级**：达到上限后，系统自动将模型切换为 GPT-5 mini，不会阻断对话
- **文字提示**：在对话界面中提示用户已切换至简化版本模型
- **无弹窗阻断**：对话可以继续，但使用的是能力较弱的模型

### 3. 是否每次对话都消耗次数
- 是的，每次发送消息都会消耗 GPT-5 额度
- GPT-5 mini 不消耗额度（无限次）

### 4. 如何引导用户付费
- 界面中提供 **"Upgrade to Plus"** 按钮
- Plus 用户：每 3 小时 **160 条** GPT-5 消息，每周 3000 条 Thinking 模式
- Pro 用户（$200/月）：所有模式不限量
- 通过模型能力差异（GPT-5 vs GPT-5 mini）形成付费动机

## 二、Claude（Anthropic）

### 1. 免费用户次数限制
- **会话制限制**：每 **5 小时** 为一个会话周期，有使用上限
- 具体消息数量**不固定**，会根据以下因素动态调整：
  - 消息长度
  - 文件附件大小
  - 当前对话长度
  - 工具使用情况（如 Research、网页搜索）
  - 模型选择
  - Artifact 创建和使用

### 2. 用完后如何提示
- **文字提示**：Claude 会在达到限制时通知用户
- **输入框禁用**：达到上限后无法继续发送消息，需等待重置
- 提示信息明确告知用户已达到使用限制

### 3. 是否每次对话都消耗次数
- 是的，所有对话消息都计入使用量
- 所有 Claude 产品界面（claude.ai、Claude Code、Claude Desktop）共用同一额度

### 4. 如何引导用户付费
- **Pro 计划**：$20/月（年付 $17/月），提供更多使用额度
- **Max 计划**：$100/月起，5x 或 20x 于 Pro 的使用量
- 付费用户在设置中可查看使用进度条（Current session / Weekly limits）
- 高峰期优先访问权

## 三、Kimi（月之暗面）

### 1. 免费用户次数限制
- **基础对话**：免费用户可使用 Kimi 的所有基础对话服务
- **长文本处理**：3 小时内 **20 次**（早期用户反馈，可能已调整）
- **Agent/深度研究功能**：有明确的调用次数限制
- 部分报道提到免费版每天限制 **10 次对话**（可能为特定功能或早期版本）

### 2. 用完后如何提示
- **弹窗提示**：用户刚注册时会弹出"开通会员享高速服务"的弹窗
- **文字提示**：达到限额后提示用户等待或开通会员
- **按钮变灰/禁用**：部分功能按钮在额度用完后变为不可点击状态

### 3. 是否每次对话都消耗次数
- 基础短文本对话：通常不消耗或消耗较少
- 长文本解析、文件分析、Agent 任务：消耗额度
- 深度研究（OK Computer）和 Kimi Code：有明确次数限制

### 4. 如何引导用户付费
- **会员三档定价**：
  - **Adagio（免费版）**：有限的使用次数
  - **Andante**：49 元/月，适量使用次数 + API 兑换券
  - **Moderato**：99 元/月，更多使用次数 + 支持并发
- 早期通过"打赏"机制过渡，后转为正式会员体系
- 付费和免费用户的主要区别在于 Agent 使用次数

## 四、文心一言（百度）

### 1. 免费用户次数限制
- **每日提问上限**：**50 次**
- **自然日零点自动重置**，不累计、不跨日延续
- **单次输入限制**：免费用户 5000 字（付费用户 2.8 万字）
- **图像生成**：免费用户每月限 10 次
- **文件上传**：单次仅支持 1 个 ≤10MB 的文本型文件

### 2. 用完后如何提示
- **系统提示文字**："对话次数达到当日上限"
- **输入框禁用**：达到 50 次后无法继续提问
- 提示用户等待次日零点重置或开通付费会员

### 3. 是否每次对话都消耗次数
- 是的，每次提问都计入 50 次的日配额
- 语义重复的内容仍计为独立请求，不会自动去重

### 4. 如何引导用户付费
- **付费会员**：不受每日 50 次限制
- 历史定价：59.9 元/月，连续包月 49.9 元/月
- 2025 年 4 月起文心一言宣布全面免费，但高级功能（如 4.0 及以上版本）可能仍需会员
- 付费入口：APP 内「我的」→「会员中心」

## 五、通义千问（阿里云）

### 1. 免费用户次数限制
- **网页版/App 个人用户**：**完全免费，所有功能无限制使用**，无任何隐形消费
- **Qwen Code（编程智能体）**：每天 **2000 次** 免费调用，无 Token 限制
- 通义千问采用"免费策略积累用户基础"的商业模式

### 2. 用完后如何提示
- 网页版/App 基础对话**无明显次数限制提示**
- Qwen Code 达到 2000 次后可能会有调用限制提示

### 3. 是否每次对话都消耗次数
- 网页版/App 基础对话：**不消耗次数**（无限制）
- Qwen Code/API 调用：按调用次数计费

### 4. 如何引导用户付费
- 目前对个人用户以**免费策略**为主
- 商业化主要通过：
  - **API 调用收费**（面向开发者）
  - **企业级服务**
  - 未来可能通过代理式 AI 能力优化淘宝购物体验实现商业化

## 七、关键发现

1. **限制策略差异**：
   - 海外产品（ChatGPT、Claude）采用"**软限制**"（自动降级模型），保证用户体验连续性
   - 国内产品（文心一言、Kimi）多采用"**硬限制**"（直接阻断输入），强制用户感知限制

2. **付费引导方式**：
   - **模型能力差异**：ChatGPT 通过 GPT-5 vs GPT-5 mini 的能力差距引导付费
   - **功能解锁**：Kimi 通过 Agent 次数限制引导付费
   - **次数解锁**：文心一言通过每日 50 次上限引导付费
   - **免费策略**：通义千问目前对个人用户完全免费，通过其他方式商业化

3. **次数重置机制**：
   - ChatGPT：5 小时滚动重置
   - Claude：5 小时会话重置
   - 文心一言：自然日零点重置
   - Kimi：根据功能不同，有小时/日/周等不同重置周期
