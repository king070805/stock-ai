# 模拟股票经验库

这个目录用于沉淀每次模拟股票后的复盘总结，让经验可以像股票分析师的 skill 一样被持续调用和迭代。

## 使用原则

- 每次模拟结束后，在本目录新增一份按日期命名的总结文件，例如 `2026-06-11.md`。
- 总结必须基于 `../信息` 目录中的网站信息、行情数据、缓存或整理后的资料。
- 记录判断依据，不只记录结论。
- 区分事实、推理、假设和待验证问题。
- 不把临时脚本、测试文件或运行日志放入本目录。

## 分析师 Skill 迭代规则

- 如果同类信号连续两次有效，提高其权重。
- 如果同类信号连续两次失效，降低其权重，并写明可能原因。
- 遇到数据缺失时，必须降低仓位或观望，不能用猜测补齐行情。
- 优先保留能解释收益和回撤的经验，删除或弱化只描述结果的经验。
- 每次总结最后都要生成下一次模拟可直接使用的规则。
